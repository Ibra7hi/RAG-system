================================================================
SCORING GUIDE FOR RAG EVALUATION
================================================================

For each question, evaluate the RAG system's response on:

  RETRIEVAL ACCURACY (Did it find the right chunk?)
    3 = Correct chunk retrieved
    2 = Partially correct / adjacent chunk
    1 = Wrong chunk retrieved
    0 = No chunk retrieved

  ANSWER CORRECTNESS (Is the answer factually accurate?)
    3 = Fully correct and specific
    2 = Partially correct or slightly vague
    1 = Incorrect but in the right area
    0 = Hallucinated / completely wrong

  HALLUCINATION CHECK (Did it refuse when it should?)
    PASS = Correctly declined to answer out-of-scope questions (Q86–Q95)
    FAIL = Fabricated an answer for out-of-scope questions

  GROUNDING (Did it cite or reference the source?)
    PASS = Response references the document / specific section
    FAIL = Response gives answer with no grounding
